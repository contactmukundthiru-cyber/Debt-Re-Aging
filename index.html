How It Works

Five simple steps from credit report to dispute letter.

1

Upload the credit report

Drop in a PDF or photo of the credit report. The tool reads the text automatically using OCR (optical character recognition). You can also paste text directly.

2

Check the extracted text

Review what the tool read from the document. If anything looks wrong (blurry scan, missing text), you can fix it here before moving on.

3

Verify the key details

The tool pulls out important fields like dates, balances, and creditor names. Double-check these are correct - especially the dates, since that's what we're analyzing.

4

See what issues were found

The tool runs 20+ checks looking for problems: dates that don't add up, debts reported too long, balances that don't match the status, and more. Each issue shows exactly why it was flagged.

5

Download your dispute letters

Get ready-to-send letters for each credit bureau. Just add the consumer's name and address, then print and mail. The letters cite specific FCRA sections and explain exactly what's wrong.

Installation

Choose the easiest option for you.

Option A: Windows (Easiest)

  1. Download the ZIP file and unzip it anywhere
  2. Double-click START_APP.bat
  3. Wait for it to install (first time takes 2-3 minutes)
  4. Browser opens automatically - you're done!
First time? You'll need Python installed. Get it free from python.org - make sure to check "Add to PATH" during install.

Option B: Command Line

  1. Open Terminal or Command Prompt
  2. Run: git clone https://github.com/contactmukundthiru-cyber/Debt-Re-Aging.git
  3. Run: cd Debt-Re-Aging
  4. Run: pip install -r requirements.txt
  5. Run: streamlit run app/main.py

Option C: Docker

  1. Install Docker Desktop
  2. Run: git clone https://github.com/contactmukundthiru-cyber/Debt-Re-Aging.git
  3. Run: cd Debt-Re-Aging && docker-compose up
  4. Open http://localhost:8501

What It Checks

The tool looks for common credit report problems automatically.

Date manipulation

Catches when collectors change dates to keep items on reports longer than 7 years

Expired debts

Flags debts past your state's statute of limitations

Duplicate accounts

Finds the same debt reported multiple times by different collectors

Balance errors

Spots paid accounts still showing balances, or fees that grew too much

Medical debt rules

Checks 2023 CFPB rules - 365-day waiting period, $500 threshold

Impossible dates

Catches dates in the future or sequences that don't make sense

Questions

Is my data safe?

Yes. Everything runs on your computer. Nothing gets uploaded anywhere. The tool works completely offline after you install it.

Is this legal advice?

No. This is an informational tool to help identify potential issues. Always have an attorney review before taking legal action.

What file types work?

PDF, PNG, JPG, and JPEG. You can also paste text directly if you've already copied it from somewhere.

Does it work on Mac?

Yes. Use Option B (command line) or Option C (Docker). The Windows batch file is just for convenience.

Who made this?
