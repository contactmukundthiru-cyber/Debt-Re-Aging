What it does

Extract

Upload a credit report PDF or image. OCR extracts dates, creditors, balances, and account details automatically.

Analyze

Nine transparent rules check for timeline inconsistencies, re-aging patterns, and statute of limitations violations.

Document

Generate dispute letters, evidence summaries, and furnisher notifications. Ready to send to credit bureaus.

Detection rules

Every flag is based on documented, auditable logic. No black boxes.

Code Rule Trigger Severity
A1 Excessive reporting Removal date exceeds 8 years from date opened High
A2 DOFD misalignment Removal date inconsistent with DOFD + 7 years High
B1 Classic re-aging Date opened 24+ months after DOFD High
B2 Missing DOFD No DOFD on collection with recent open date Medium
C1 Bureau inconsistency Different removal dates across bureaus Medium
D1 SOL expired Account reported after statute of limitations High
D2 Impossible timeline DOFD predates account open date High
D3 Ancient account Account age exceeds 15 years Medium
E1 Balance irregularity Balance increased after charge-off date High

Installation

Docker (recommended)

All dependencies included. Requires Docker Desktop.

# Clone and run git clone https://github.com/contactmukundthiru-cyber/Debt-Re-Aging.git cd Debt-Re-Aging docker-compose up # Open http://localhost:8501

Python

Native installation for custom setups.

# Install dependencies git clone https://github.com/contactmukundthiru-cyber/Debt-Re-Aging.git cd Debt-Re-Aging pip install -r requirements.txt # Run streamlit run app/main.py

Common questions

What is debt re-aging?

An illegal practice where debt collectors manipulate dates to extend how long negative information appears on credit reports, violating the Fair Credit Reporting Act.

Is my data secure?

Yes. All processing happens locally. Nothing is uploaded anywhere. The tool works completely offline after installation.

Who is this for?

Legal aid organizations, law school clinics, consumer advocacy groups, and pro bono attorneys helping clients dispute credit report errors.

Is this legal advice?

No. This is an informational tool. Always consult a qualified attorney before taking legal action.

What file formats work?

PDF, PNG, JPG, and JPEG. The tool uses OCR to extract text from scanned documents.
