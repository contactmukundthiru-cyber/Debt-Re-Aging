# Marketing images

## Open Graph / share image

Add a share image here for social links (e.g. `og.png`).

- **Recommended size**: 1200×630 px
- **Format**: PNG or JPG
- **Usage**: In `index.html`, set:
  ```html
  <meta property="og:image" content="https://contactmukundthiru-cyber.github.io/Debt-Re-Aging/img/og.png">
  <meta name="twitter:image" content="https://contactmukundthiru-cyber.github.io/Debt-Re-Aging/img/og.png">
  ```
  Use your actual site URL (e.g. GitHub Pages or custom domain); relative paths do not work for social crawlers.

If you don’t add an image, leave the meta tags empty and social previews will fall back to title/description only.
